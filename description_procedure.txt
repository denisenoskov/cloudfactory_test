на MySQL при вызове процедуры,
 - все поля обязательны к заполнению(ограничения MYSQL)
входные параметры:
- название идеи , проверяется точное вхождение в строку в любом месте, чтобы не фильтровать по названию, необходимо передать NULL
- описание идеи, подсчитывается вхождение каждого слова в описании, чем больше тем выше рейтинг по которому впоследствии отсортирует от большего к меньшему записи
- наименьший целевой доход, задается  числом с плавающей запятой в %, чтобы не фильтровать - необходимо передать NULL
- наибольший целевой доход, задается  числом с плавающей запятой в процентах,  чтобы не фильтровать - необходимо передать NULL
- источник идеи , проверяется точное вхождение в строку в любом месте, чтобы не фильтровать по названию, необходимо передать NULL
выходные параметры:
- Порядковый номер в таблице, начинается с 1
- Титульное имя идеи, заголовок
- Подробное описание идеи
- Цена акций в момент публикации идеи, не может быть отрицательной, выражена в рубля
- Цена акций на текущий момент, не может быть отрицательной, выражена в рублях
- Целевая цена акций планируемая на дату завершения срока идеи, не может быть отрицательной, выражена в рублях
- Дата публикации идеи
- Целевая дата для идеи, когда цена станет целевой
- Титульное имя источник идеи
- Оценка количества совпадений в описании с фильтром по description, применима когда был фильтр по описанию, иначе 0
- Планируемая прибыль в процентах, рассчитывается как отношение целевой цены к стартовой цене минус 100%
